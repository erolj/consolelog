Npm module to insert default console message on each page.

## Install

```node
npm install @erolj/consolelog
```

## Usage

```js
const demo = require('@erolj/consolelog');
const text = "Your console message goes here"

demo(text);
```

## Output

```bash
Your console message goes here
```
